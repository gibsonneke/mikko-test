# Mikko-test
Required: Git, Composer, PHP 5.5+ CLI

1. Clone the code from GitHub:

    # git clone https://github.com/gibsonneke/mikko-test.git

2. Run Composer:

    # composer install

3. Generate the CSV file as payments.csv

    #  ./generate-payments.php
